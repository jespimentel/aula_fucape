# O Poder de IA e do Python na Coleta e Tratamento de Dados Judiciais

Material da aula ministrada na FUCAPE em junho de 2026.

## Conteúdo

- **Transcrição de audiências judiciais** — pipeline com FFmpeg, Word Online e modelos de IA (Whisper)
- **Análise de dados da API do DataJud** — consultas ao CNJ via Elasticsearch

## Notebooks

- [`transcricao_ffmpeg_whisper.ipynb`](transcricao_ffmpeg_whisper.ipynb) — transcrição e tratamento de áudio
- [`datajud_tjma.ipynb`](datajud_tjma.ipynb) — coleta e análise de dados do TJMA via API do DataJud
